Exercise 1: Implementing Your Own Callbacks

Task: Expand on the example provided. Implement two more callback functions: multiply and divide. Use the operate function to perform these operations on two numbers.

1. Create a `multiply` function that takes two numbers and returns their product. 2. Create a `divide` function that takes two numbers and returns their quotient. 3. Use the `operate` function to perform multiplication and division, logging the results to the console.
